Packagecom.platogo.flow.core
Classpublic final class FlowInternal
InheritanceFlowInternal Object

FlowInternal registers instanciated managers and provides access for classes that use multiple managers. E.g.: The CommandScreen Class is a Screen and needs access to the Screenmanager , but also to the Commandmanager (but not to the DataManager and the LogManager).
